# Fake Captive Portal using ESP8266

This project implements a **Fake Captive Portal** using the **ESP8266**. It creates a fake Wi-Fi Access Point without internet access and displays a fake captive portal webpage to collect user data such as email and password.

## Features
- Create a fake Wi-Fi Access Point using the ESP8266
- Display a webpage with a form to input email and password.
- Capture user-submitted data and display it in the Serial Monitor.
- Redirect all domain requests to the main page (Captive Portal).

## Requirements
- **ESP8266 based modules**
- **Arduino IDE**
- **ESP8266 Board Package**
